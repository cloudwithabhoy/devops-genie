# Docker Project Ideas

> Hands-on projects to build real Docker skills.

## Beginner

- **Containerise a Web App** — Write a Dockerfile for a simple Node.js or Python Flask app and run it locally.
- **Docker Compose Stack** — Use Compose to run a web app + database (e.g., WordPress + MySQL).

## Intermediate

<!-- Add intermediate project ideas here -->

- **Multi-Stage Build** — Reduce an image size by 70%+ using a multi-stage Dockerfile.
- **Private Registry** — Set up a local Docker registry and push/pull images from it.
- **Healthcheck & Restart Policies** — Configure health checks and auto-restart for a containerised service.

## Advanced

<!-- Add advanced project ideas here -->

- **Docker Swarm Cluster** — Deploy a multi-node Swarm cluster and run a replicated service with rolling updates.
- **Rootless Docker** — Configure and run Docker in rootless mode for improved security.
- **Custom Docker Network** — Build a multi-container app with isolated frontend and backend networks.

## Project Template

For each project, document:
1. **Goal** — What you're building and why
2. **Dockerfile / Compose file** — Full file content
3. **Steps** — How to build and run
4. **Verification** — How to confirm it works
5. **Key Learnings** — What concepts this demonstrates
