# AWS Project Ideas

> Hands-on projects to build real AWS experience.

## Beginner

<!-- Add beginner project ideas here -->

- **Static Website on S3** — Host a static HTML/CSS site on S3 with CloudFront CDN and a custom domain via Route 53.
- **EC2 Web Server** — Launch an EC2 instance, install Nginx, and serve a simple web page.

## Intermediate

<!-- Add intermediate project ideas here -->

- **Serverless REST API** — Build a REST API using API Gateway + Lambda + DynamoDB.
- **Auto-Scaling Web App** — Deploy an app behind an ALB with Auto Scaling Groups across multiple AZs.
- **CI/CD Pipeline** — Use CodePipeline + CodeBuild + CodeDeploy to automate deployments.

## Advanced

<!-- Add advanced project ideas here -->

- **Multi-Region Architecture** — Build a globally distributed application with Route 53 failover routing.
- **EKS Cluster** — Deploy a containerized microservices application on EKS with Helm charts.
- **Landing Zone** — Set up a multi-account AWS organization with Control Tower and Service Control Policies.

## Project Template

For each project, document:
1. **Goal** — What you're building and why
2. **Architecture Diagram** — Draw.io or ASCII diagram
3. **Services Used** — List of AWS services
4. **Steps** — Step-by-step implementation guide
5. **Cost Estimate** — Approximate monthly cost
6. **Cleanup** — How to tear down resources
