## Key Topic Areas

### Bash
- Variables, arrays, and arithmetic
- Conditionals and loops
- Functions
- Input/output and pipelines
- Error handling (`set -euo pipefail`)
- String manipulation
- Working with files and directories
- Cron and scheduling

### Python
- Variables, data types, and control flow
- Functions, modules, and packages
- File I/O and os/sys modules
- Working with APIs (requests)
- Parsing JSON/YAML/CSV
- subprocess for shell commands
- boto3 for AWS automation
- CLI tools with argparse / click
