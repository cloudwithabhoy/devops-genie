## Key Topic Areas

### Terraform
- HCL syntax: providers, resources, variables, outputs, locals
- State management (local, remote, S3 backend)
- Modules and the Terraform Registry
- Workspaces and environment management
- Import and drift detection

### Ansible
- Inventory, playbooks, tasks, roles
- Variables, facts, templates (Jinja2)
- Handlers, conditions, loops
- Vault for secrets
- Ansible Galaxy

### General IaC Concepts
- Declarative vs imperative IaC
- Idempotency
- Immutable infrastructure
- Policy as Code (OPA, Checkov)
