# Linux Project Ideas

> Hands-on projects to build real Linux administration skills.

## Beginner

- **User Management Script** — Write a Bash script to automate creating users, setting passwords, and assigning groups.
- **Log Analyser** — Parse `/var/log/auth.log` or `/var/log/syslog` to find failed login attempts.

## Intermediate

<!-- Add intermediate project ideas here -->

- **Automated Backup Script** — Use `rsync` and cron to back up directories to a remote server on a schedule.
- **System Health Dashboard** — Build a script that outputs CPU, memory, disk, and network stats in a formatted report.
- **Web Server Setup** — Install and configure Nginx or Apache with virtual hosts and Let's Encrypt SSL.

## Advanced

<!-- Add advanced project ideas here -->

- **Hardened Linux Server** — Configure a fresh Ubuntu server following CIS benchmark recommendations.
- **Custom Kernel Module** — Write and load a simple Linux kernel module.
- **LVM Storage Setup** — Configure LVM with logical volumes, snapshots, and online resize.

## Project Template

For each project, document:
1. **Goal** — What you're building and why
2. **Environment** — OS version, required packages
3. **Steps** — Step-by-step implementation
4. **Verification** — How to confirm it works
5. **Cleanup** — How to undo changes
